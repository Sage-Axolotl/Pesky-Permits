Adds permits inspired by Hermitcraft.

To obtain them, give yourself an item with one of the following models:

- sage:redstone_permit
- sage:emerald_permit
- sage:lapis_permit
- sage:diamond_permit
- sage:quartz_permit
- sage:amethyst_permit
- sage:iron_permit
- sage:copper_permit
- sage:gold_permit

Example command:
```
/give @s minecraft:paper[item_model="sage:amethyst_permit"]
```
